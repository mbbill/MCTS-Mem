- A node's evidence is an inline `## Facts` section; a fact with a substantial body — a
  measurement table, a recovered document, a quoted interview — graduates to a file in a
  sibling `<name>.fact/` folder linked from the section.

- A fact's origin is its position in the tree; there is no origin field.

## Moves

- 2026-06-19 replaced [[evidence-pool]]: a flat pool of fact files linked by an `origin:`
  field forced navigating a central directory by that field rather than an option's own path, and let
  file names balloon, while
  co-located inline facts keep a node self-contained and travel with it on a pivot (sourced).
